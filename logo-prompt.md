# Logo Design Prompts — HUP Corner

## Design Vibe
- Swiss Grid editorial, academic journal, warm minimalism
- Palette: Burgundy (#8A3D38), Olive (#7C8C68), Paper (#F7F4EE), Charcoal (#171717)
- Typography: Playfair Display (serif), Inter (sans), JetBrains Mono
- Context: HUP Corner — Thư viện tri thức học thuật (Hanoi University of Pharmacy)

---

## Prompt 1 — Trừu tượng, có chữ ẩn

**Midjourney:**
```
Minimalist abstract emblem for "HUP Corner" — an academic knowledge library. 
A single elegant symbol combining a stylized open book with corner-folded page transforming into the letter "H" or a stacked geometric monogram "HC". 
Swiss-grid precision, editorial sophistication. 
Color palette: deep burgundy (#8A3D38) and warm paper (#F7F4EE) background, subtle olive (#7C8C68) accent.
Clean vector lines, slight texture like letterpress printing, zero text, abstract and timeless.
Aspect ratio 1:1 — style: flat vector with subtle paper grain texture
```

**DALL-E:**
```
Minimalist abstract logo mark for an academic pharmacy knowledge platform. A geometric monogram combining an open book with a folded corner, forming an abstract "H" shape. Swiss design style, editorial refinement. Colors: warm paper beige background, dark charcoal linework, deep burgundy accent. No text. Clean vector illustration with slight paper texture. 1:1 aspect ratio.
```

## Prompt 2 — Hoàn toàn không text, thuần symbol

**Midjourney:**
```
Abstract editorial monogram for an academic pharmaceutical library brand. 
A minimalist geometric icon: folded book page transforming into a mortar-and-pestle silhouette, or an abstract "H" constructed from stacked horizontal rules and one vertical stroke resembling a bookshelf. 
No text. Pure symbol. Swiss grid precision, letterpress texture, white space dominance. 
Monochromatic palette: warm charcoal ink (#171717) on uncoated paper (#F7F4EE), with a single micro dot of burgundy (#8A3D38) as accent. 
Woodcut or linocut print aesthetic, high contrast, timeless scholarly feel. 
1:1 square, clean vector, no gradients, no shadows.
```

**DALL-E:**
```
Abstract minimal logo mark for a pharmacy school's academic library. Zero text. A single geometric symbol: either an open book with pages forming a subtle mortar outline, or a minimalist "H" built from just two horizontal lines and one vertical — like a minimalist bookshelf. Black ink on warm cream paper. One tiny burgundy circle as the only color. Linocut print style, Swiss editorial layout. Very minimalist, very academic. 1:1 vector icon.
```
